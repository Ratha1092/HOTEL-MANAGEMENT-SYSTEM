
FINAL HOTEL MANAGEMENT SYSTEM (ONE HOTEL)

HOW TO RUN:
1. Install XAMPP
2. Copy folder to: C:/xampp/htdocs/HotelSystem
3. Start Apache & MySQL
4. Import database/hotel_db.sql via phpMyAdmin
5. Open:
   - Frontend: http://localhost/HotelSystem/public/index.php
   - Admin: http://localhost/HotelSystem/admin/auth/login.php
